Darkmode is pretty trendy in UI/UX right now—and for good reason! I oftentimes am working on this website into the wee hours of the morning, and while I love my bright colors, it can feel like staring into the sun when the lights are off.

We've come to expect darkmode from websites. But I believe us web designers are capable of so much more!

So I would like to purpose a few more potential toggles that I think many will come to see as a necessity.

Ever wanted to create a website, but didn't know where to start?

Well start here. In this series of tutorials, we will cover how to first build a static website using just HTML and CSS. Then we will add in bootstrap, which is a CSS framework. Then we will cover how to add in a backend using flask. This will make our website signficantly easier to maintain and update. Finally, we will go over how to host your website on a raspberry pi, so that other people can access your website!
