Get Wikipedia Data with getwiki

Getting Wikipedia data into R can be a breeze with getwiki . The purpose of this package is to import Wikipedia articles into R, quickly and easily. getwiki will return articles in a format that plays nice with tidytext, simplifying your NLP workflow. Previously, the easiest way to import the text of a Wikipedia article was to highlight everything and then copy paste it into a character vector. Check out the github page!

Automated Deck Building with Xaringan

Do you hate updating decks each week?

Then automate it using xaringan! The xaringan package extends R Markdown, so that it can create HTML 5 presentations based on the remark.js framework. Basically, you can create decks with R Markdown. You can inject the results of R code right into the presentation and customize the appearance using CSS. Personally, I have used this technology to automated two weekly decks, saving about one workday's worth of work each week.

Natural Language Processing in R on Udemy

I am happy to announce that my course "Natural Language Processing in R for Beginners" is published on Udemy! It covers how to import text data into R, how to clean it with packages like tidytext and tm, and how to analyze it with methods like topic modeling and sentiment analysis.
