Analytics

Analytics is the discpline of getting insights out of data, and these insights are usually delivered in the form of a data product. A data product can be anything from a model, to an automation, to a report of summary statistics, to a web app.

Thus far in my career, I have made data products ranging from simple queries delivered in excel reports and powerpoint decks, to web apps built in shiny, to automations built in r markdown, and to anomaly detection models using time series decomposition.

Automated Deck Building with Xaringan

Xaringan package extends R Markdown, so that it can create HTML 5 presentations based on the remark.js framework. Basically, you can create decks with R Markdown. You can inject the results of R code right into the presentation and customize the appearance using CSS. Personally, I have used this technology to automated two weekly decks, saving about one workday's worth of work each week.
